# myapp
Hazme un GridView de 2 columnas (14 servicios en total). Quiero que cada tarjeta se vea súper limpia y moderna, con bordes redondeados y una sombra sutil.

Debajo de la imagen, ponme tres filas:

El nombre del tratamiento (un título con una fuente con personalidad).

Una descripción corta (subtítulo).

Las típicas 5 estrellitas doradas.

Usa una paleta de colores frescos (azules, blancos, turquesas) y que las tipografías no sean las de siempre. Ah, y nada de imágenes de internet, que el código use assets locales. ¡Que el diseño se vea de alta calidad!"
